# Static scene backgrounds

Add webp images here for instant-loading quest scenes. When present, they show immediately instead of waiting for DALL-E. When absent, the theme gradient is used.

Expected files:
- minecraft-builder.webp
- kitchen-scientist.webp
- playground-lab.webp
- pattern-detective.webp
- nature-explorer.webp
- adventures.webp

Recommended: ~1024x1024 or 16:9, kid-friendly illustration per chapter theme.
